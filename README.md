# infinite-app

### Server
1) Выполнить скрипт из файла init-db.sql
2) Перейти в директорию server командой: 
    ```shell
    cd server
    ```
3) Выполнить команду:
    ```shell
    npm install
    ```
4) Создать файл .env по примеру файла .env.example и заполнить верными данными
5) Выполнить команду:
    ```shell
    npm start
    ```

### Client
1) Перейти в директорию client командой:
    ```shell
    cd client
    ```
2) Выполнить команду:
    ```shell
    npm install
    ```
3) Выполнить команду:
    ```shell
    npm start
    ```
